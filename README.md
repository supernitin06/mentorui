# Mentora - Neon Mentorship Platform

A simplified frontend for a mentorship platform where parents, students, and mentors interact. Built with **Next.js**, **Tailwind CSS v4**, and **Framer Motion**.

## ✨ Features
- **Vibrant Dark Mode**: Sleek UI with neon orange, blue, and purple accents.
- **Glassmorphism**: Modern glass-effect components throughout the app.
- **Role-Based Views**: Tailored dashboards for Parents and Mentors.
- **Dynamic Animations**: Smooth transitions using Framer Motion.
- **Parent Controls**: Student management and lesson booking UI.
- **Mentor Tools**: Course creation and session management UI.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure
- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI and section-specific components.
- `src/lib`: Mock data and utility functions.
- `src/hooks`: Custom React hooks (if any).

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
